## Task 1: Quantum Computing

This task demonstrates basic quantum circuits using Cirq:

1. **5-Qubit Circuit**
   - Hadamard gates on all qubits
   - Chain of CNOTs (0→1, 1→2, 2→3, 3→4)
   - SWAP between qubits 0 and 4
   - X rotation (π/2) on qubit 2

2. **SWAP Test Circuits**
   - SWAP test between single-qubit and 2-qubit states
   - Measures overlap (fidelity) of quantum states
   - Histogram shows ancilla measurement results

All scripts are in this folder and can be simulated locally using Cirq.
